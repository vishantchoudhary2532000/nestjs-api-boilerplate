## Overview
Production-ready boilerplate for building scalable REST APIs using NestJS.

## Tech Stack
NestJS, TypeScript, MongoDB (Mongoose), JWT, Winston

## Features
- Auth (JWT)
- User CRUD handling
- Modular architecture
- Centralized error handling & validation
- Role-based access control (RBAC)
- Environment config

## Setup
npm install  
cp .env.example .env  
npm run start:dev
